About the Project

This web application provides interactive visualizations of global climate data from various cities around the world. It allows users to explore and analyze CO2 levels and temperatures of different locations through an interactive map. The goal of this project is to help users understand climate patterns and the relationship between CO2 levels and global temperatures.

The data is fetched from a backend system that sources real-time information to provide accurate climate data. This project was created as part of a climate awareness initiative.

We strive to make climate data accessible and engaging through a user-friendly interface and seamless functionality.
